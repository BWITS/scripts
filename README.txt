Contents:

A miscellaneous collection of scripts.

notes.pl:

  - A script that allows a PSE to overwrite specific slides in the 
    Fundamentals course with presenter notes of his/her own.

setup.sh:

  - A script to automatically set up the classroom environment at
    the beginning of a Fundamentals class.

org_chart.rb:

  - Creates a DOT graph of the company's org chart.

tip.rb:

  - calculates a tip amount.
